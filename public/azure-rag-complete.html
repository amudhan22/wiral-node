üìö API Reference

POST/api/azure-rag/upload

Complete Azure RAG processing - Upload, extract, embed, and store

curl -X POST http://localhost:3009/api/azure-rag/upload \\
  -F "files=@document.pdf" \\
  -F "account_id=1" \\
  -F "title=My Document" \\
  -F "description=Important document"

POST/api/azure-rag/search

Search documents using AI embeddings

curl -X POST http://localhost:3009/api/azure-rag/search \\
  -H "Content-Type: application/json" \\
  -d '{
    "account_id": 1,
    "query": "pricing policy",
    "limit": 5
  }'

GET/api/azure-rag/files/:accountId

List all files for an account

curl -X GET http://localhost:3009/api/azure-rag/files/1

GET/api/azure-rag/health

Check system health and Azure connection

curl -X GET http://localhost:3009/api/azure-rag/health
